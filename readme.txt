Enables Post Thumbnails support.

== Description ==

This light-weight plugin enables Post Thumbnails support. To display thumbnails, use: the_post_thumbnail();

== Installation ==

1. Install the plugin in the regular way
1. Activate the plugin through the 'Plugins' menu in WordPress
